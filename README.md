# 🚀 POCO X6 — Landing Page de Pré-Lançamento

> Acesse a página ao vivo: [herbertis.github.io/pre-lancamento-poco-x6-page](https://herbertis.github.io/pre-lancamento-poco-x6-page/)

---

## Sobre o Projeto

Landing page de pré-lançamento desenvolvida para o **POCO X6**, criada para gerar expectativa e engajamento antes da chegada oficial do produto ao mercado brasileiro.

A página foi construída para uso nos **canais oficiais da Xiaomi Brasil (Mi Brasil)**, seguindo os padrões visuais das campanhas da marca — com foco em experiência imersiva, narrativa de produto e forte apelo visual.

---

## Funcionalidades

- **Hero Section de Pré-Lançamento** — destaque visual do produto com comunicação voltada à geração de expectativa
- **Storytelling de Produto** — narrativa que apresenta os principais diferenciais da linha POCO
- **Layout de Campanha** — interface inspirada em páginas de lançamento de grandes marcas de tecnologia
- **Design Responsivo** — adaptado para Mobile, Tablet e Desktop
- **Seções Visuais de Destaque** — blocos que reforçam inovação, performance e design do dispositivo
- **Performance Front-end** — estrutura leve e otimizada para carregamento rápido

---

## Design & UX

Projeto desenvolvido com a visão de unir **Design e Desenvolvimento Front-end**, criando uma experiência visual capaz de gerar impacto antes do lançamento oficial.

Princípios aplicados:

- **Hierarquia Visual** — destaque estratégico das seções promocionais
- **Mobile First** — interface pensada inicialmente para dispositivos móveis
- **Estética de Marca** — layout alinhado com a identidade visual da POCO
- **Design Orientado à Campanha** — estrutura construída para gerar antecipação no usuário

---

## Valor de Negócio

Landing pages de pré-lançamento são peças-chave em estratégias de marketing digital para produtos tecnológicos. Este projeto entrega:

- **Geração de expectativa** para o lançamento oficial
- **Engajamento visual** que estimula a exploração da página
- **Fortalecimento de marca** com comunicação alinhada à POCO
- **Base para campanhas digitais** e divulgação em redes sociais
